# **App Name**: WebJackTimer

## Core Features:

- Symbol Input: Allow users to input symbols using the keyboard to represent elements of a timing diagram.
- Diagram Rendering: Render the input symbols into a visual timing diagram on the web page.
- Customization: Provide options for users to customize the appearance of the timing diagram (e.g., colors, spacing).
- Diagram Saving: Enable users to save and load timing diagrams for later use. Store the diagrams in the Firestore database
- Symbol Suggestions: Generative AI tool to suggest appropriate symbols based on the current context of the diagram and user input.
- Error checking: Generative AI tool that analyzes the diagram to identify potential timing errors or inconsistencies, using a tool for reasoning about the diagram's structure.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) to convey a sense of calm and focus.
- Background color: Very light blue (#EAF2FF), almost white, providing a clean backdrop.
- Accent color: Muted purple (#A394E0) for interactive elements and subtle highlights.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean and readable user interface.
- Use minimalist icons to represent diagram elements and functions (save, load, customize).
- Clean, spacious layout with the diagram taking center stage. Provide clear controls for symbol input and customization.
- Subtle animations for diagram updates and user interactions, providing visual feedback without distraction.
